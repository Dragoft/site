## 介绍
>`S E K A I`「せかい」是一款使用 Cloudflare Pages 部署的响应式博客主题。<br /><br />UI 设计受到了 [kotobank.jp/](https://kotobank.jp/) 的启发，主体代码由 `Tatsuno Yuu` 编写。<br />为保证网站的独特性，此主题 **暂不授权任何形式的使用** 但你仍可以学习它的源码（不嫌弃我糟糕的代码就行）

<br />

## 主题部署
>由于对源码使用的限制，这里不提供主题的部署方法。SQL 数据库结构保密。<br />P.S.: 此 Github 账号长期离线，请通过其他方式联系作者。

<br />

## 使用的技术
* **Cloudflare 服务**
	* Pages
	* Pages Function
	* D1 数据库
	* Cloudflare Web Analytics
* **前端**
	* iframe 博客页面嵌套
	* JavaScript postMessage 页面通信
	* CSS 响应式设计
