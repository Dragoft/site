Safebooru 介绍

—— 用了很久，挺不错的一个网站，特地为它写了这篇文章。

Safebooru is a offspring of Gelbooru hosted by a different person but running the same software. Safebooru has been created after danbooru dropped the project.
—— Safebooru
Safebooru 像是 Danbooru 的一个分支。
以二次元作品为主，但偶尔也可以找到风景、动物、食物等其他主题的图片。

相比与 Danbooru，它更加的注重 “安全”，也就是说里面没有那些很 ... 嗯 ...... 的图像。目前在国内可以正常访问。

链接： safebooru.org/


和 Danbooru 的区别

相比之下，Danbooru 是个更加开放的二次元社区，基本上可以上传任何的图片。 个人认为，Safebooru 上的氛围要好很多。这就是我推荐它的原因。

Safebooru 细节

更新速度

一个昵称为 “Danbooru” 的账号会以 30 min / 次 的速度上传图片。一次上传大约 40 张 “安全” 的图像。资源相对丰富。但其他用户的上传量较少。（估计都去 Danbooru 了）

id

虽然几乎全部 Safebooru 收录的图片在 Danbooru 都可以找到，但这并不意味着两个网站使用相同的 ID。

pid

Safebooru 的翻页系统以 pid 参数表示当前的页数。pid = ( 实际页码 - 1 ) * 40，这一点与 Danbooru 也有所不同。感觉 pid 越大，页面加载起来越困难。

账号

无须登录，便可以浏览 Safebooru 的大部分内容（包括给图片打分）。登录后便可以评论、收藏图片。出乎意料的是，账号注册的过程异常的简单！

审核

If you happen to find an image that does not follow policy then click on flag for delete link. This way the mods can easily delete the images.
—— Safebooru Forum
最令我感兴趣的是，Safebooru 有一套自己的 “审核机制”。

Safebooru 是二次元世界的 “净土”。任何人都可以上传图片，但同时任何人都可以点击页面上的 Flag for deletion，提示管理员删除违规的图像。因此，Safebooru 上几乎都是很正经的图像。
