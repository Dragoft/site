雾中高地

雨暂时停了。抓住这个机会，我出去了。
路面仍然是湿的，天空中又开始飘起了毛毛雨。整个县城笼罩在一片抑郁的灰色中。我曾经在书上读到过对苏格兰高地的描述，旁边还好心地配了一张图：雾蒙蒙的，低矮的植被，远处山脉的轮廓，像极了今天的样子。

周末到了，商场是学生们的天下。有的来逛商场，有的只顾着刷手机，还有的手牵着手，谈情说爱。不得不承认，我是真的比不过他们。对文艺圈一窍不通，玩不来王者、原神，连奶茶也不会买 —— 在我的印象中奶茶永远只有 珍珠 椰果 两种口味。被孤立也是有原因的！

很不喜欢这种热闹的场面，一剪完头，我便回去了。雨点划过车窗，公交车还是和来时一样拥挤。

魔法的高深之处

我一直不喜欢 python 这门语言。缩进死板，调试困难。但这是我唯一入门的后端语言。

我通过一次所谓的编程比赛接触到了 python 。至于比赛本身，则真的很让我失望。与其说是在考察编程的能力，还不如说是在考验考生的海马体：
“https 和 http 有什么区别？”
“SQL 数据库中的选择语句是什么？”
“在 python 中，以下哪个模块用于进行网络请求？”

外面仍在下着雨。看着以前用 tkinter 库做出的简陋窗口，我下定了决心，准备探究在 win7 系统做出 win11 风格窗口的方法。魔法的高深之处便在于 PyQt5 库 的调用。与「知识付费」斗争了一下午后，我勉强看懂了大佬们的代码。这就是最终的成果。


测试使用的图片源于网络，侵删。

写了 200 多行的代码，仅为打开一个窗口。这也许就是魔法的高深之处吧。

夏洛克 · 福尔摩斯先生

晚餐是一袋海鲜味的方便面。拉面、虾球、蟹棒，我从初中吃到现在，味道一直没有变过。

家里有一本「福尔摩斯探案集」，以前 1 元钱买的，今天终于读完了。大多数人喜欢在厕所里看手机，我却喜欢看这个。柯南 · 道尔的文采胜过了封面的劣质，鼓励着我继续读下去。

我这个人只剩下个脑子，其他的器官只是脑子的附属品。
—— Holmers
虽然没有福尔摩斯天才般的办案能力，但也和他一样，我对自己专业以外的事情一概不感兴趣。
我和他们基本上没有交集。谈追星，谈动态，谈游戏，我几乎一窍不通。谈我的专业，他们也几乎一窍不通。事实证明，专业太冷门是会被孤立的！

好在我已经习以为常了。我自己也得承担一部分责任，明明很多话的人，却可以被测出有 86% 的内向。华生 这样的朋友不是那么好找的。虽然不能理解福尔摩斯，但他始终默默支持着他。

不想写了。天已经黑了。外面还在下着小雨，准备上床睡觉了。
